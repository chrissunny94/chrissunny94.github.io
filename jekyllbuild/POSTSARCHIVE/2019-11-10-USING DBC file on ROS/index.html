DBC file is a proprietary format that describes the data over a CAN bus. In this article, you will learn the basic syntax of a DBC file that defines up to 8 bytes of CAN message data. A lot of CAN bus related tools can read the DBC file and display values next to each “signal” that you define in the DBC file.

The database .dbc file contains definitions of CAN messages and signals. Using the information defined in the database file, you can look up message and signal information, and build messages.
